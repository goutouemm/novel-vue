# 小说影音系统
# **项目技术**

- 前端技术：Vue, vuex, element-plus, node, axios
- 后端技术：Spring Boot, Spring Security, Mybatis-plus(大佬可改为Spring Jpa Data), elasticsearch, redis
- 开发环境：Java 13, MySQL 5.7, Maven 3.8, IDEA 2021

## 后台管理系统
1. 登录注册
